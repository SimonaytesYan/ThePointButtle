# The point battle

Шутер для двух игроков с захватом точек

## Как запускать

[Сайт с игрой](threadjava800.github.io/ThePointBattle/)

1. Зайдите на сайт, дождитесь загрузки экрана.
2. В правом верхнем углу будет два текстовых поля.
Чтобы создать сессию, введите название комнаты (любое) и нажмите Create.
3. После небольшого ожидания вас перекинет в игру и в правом верхнем углу появится код лобби (для подключения клиента).
4. Теперь этот код нужно ввести во второе поле с клиента и нажать Join.
5. Опять же, после небольшого ожидания, второй игрок подключится к тому же лобби.
